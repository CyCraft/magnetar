# Magnetar 🌟

A framework-agnostic syncing solution that auto-connects any DB/API with your local data store and has optimistic-UI built in.

```sh
npm i magnetar
```

## Meet the family

- [Magnetar](https://github.com/cycraft/magnetar) 🌟
  > A framework-agnostic syncing solution that auto-connects any DB/API with your local data store and has optimistic-UI built in
